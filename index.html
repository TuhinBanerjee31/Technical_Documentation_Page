Introduction

HTML Stands for HyperText Markup Language, where HyperText stands for Link between web pages and Markup Language means Text between tags that define the structure.HTML is a markup language that is used to create web pages. It defines how the web page looks and how to display content with the help of elements. It forms or defines the structure of our Web Page.

CSS stands for Cascading Style Sheets. It is the language for describing the presentation of Web pages, including colours, layout, and fonts, thus making our web pages presentable to the users.CSS is designed to make style sheets for the web. It is independent of HTML and can be used with any XML-based markup language.

Html Basics

Html consist of many kind of functional tags which represents user input statements and make it work accordingly. Some of the basic tags are (all the tags will be inside "< >") :-

  • "html" ~ Represents the entire HTML documentation.
  • "head" ~ Represents the prologue of the HTML documentation.
  • "body" ~ Represents the informative content part in the HTML documentation.
  • "title" ~ Represents the title of the HTML document.
  • "h1" ~ Represents first-level heading, which is actually larger in size .It also consists of an series till "h6" in ascending precedence order.
  • "p" ~ Represents the paragraph and mainly used for creating new paragraph.<\li>
  • "br" ~ This tag is used for breaking line / changing line.
  • "hr" ~ Creates a horizontal line on the page.
  • "a href =" ~ This tag is used for connecting an element with another element through links.
  • "strong" ~ Bolds the text for highlighting the importance.
  • "form" ~ Represents an HTML form for user input.
  • "input" ~ Represents input control.
  • "img" ~ Represents images.
  • "nav" ~ Represents the navigation links.
  • "ul" ~ Represents an unordered list.
  • "ol" ~ Represents an ordered list.
  • "li" ~ Represents a list item.
  • "div" ~ Represents an section in the documentation. Basically it is used as an marker tag.
  • "footer" ~ Represents a footer for a document or section.
  • "main" ~ Represents the main content of a document.
Css Basics

Css consist of several properties which helps to design the html works. Some of the basic properties are:-

  • "background" ~ A shorthand property for accessing all the background.
  • "background-color" ~ Specifies the background color of an element.
  • "border" ~ A shorthand property for border-width, border-style and border-color.
  • "border-radius" ~ A shorthand property for adjusting radius of the border.
  • "bottom" ~ Sets the elements position, from the bottom of its parent element.
  • "color" ~ Sets the color of text.
  • "column-gap" ~ Specifies the gap between the columns.
  • "flex" ~ A shorthand property for the flex-grow, flex-shrink, and the flex-basis properties.
  • "font-size" ~ Specifies the font size of text.
  • "gap" ~ A shorthand property for the row-gap and the column-gap properties.
  • "height" ~ Sets the height of an element.
  • "justify-content" ~ Specifies the alignment between the items inside a flexible container when the items do not use all available space.
  • "@keyframes" ~ Specifies the animation code.
  • "letter-spacing" ~ Increases or decreases the space between characters in a text.
  • "line-height" ~ Sets the line height.
  • "margin" ~ Sets all the margin properties in one declaration.
  • "margin-bottom" ~ Sets the bottom margin of an element.
  • "margin-left" ~ Sets the left margin of an element.
  • "margin-right" ~ Sets the right margin of an element.
  • "margin-top" ~ Sets the top margin of an element.
  • "opacity" ~ Sets the opacity level for an element.
  • "padding" ~ A shorthand property for all the padding.
  • "row-gap" ~ Specifies the gap between the grid rows.
  • "text-align" ~ Specifies the horizontal alignment of text.
  • "animation" ~ A shorthand property for all the animation.
